A fun game.

Second one from the blooDot series to be published after Solid Air.  
  
All sources are on dev, no code on main.  
  
This one is done as a recreational programming side project, to level up skill and experience before the final masterpiece.
  
It uses a custom-built indie engine built on top of SDL3 and box2d, and has Lua+Sol2 as its script engine to compile the level scripts and interpret the firing trajectory commands.
